# Activos de marca — Edificio Dr. Domagk 2 (DD²)

Estos PNG son **placeholders** generados automáticamente
(`node scripts/gen-brand-placeholders.mjs`). Sustitúyelos por los oficiales
**manteniendo exactamente los mismos nombres** y la UI los recogerá sin tocar código.

| Archivo | Uso en la app |
|---|---|
| `marca_beige.png` | Cabecera del sidebar (sobre fondo bronce) |
| `marca_bronce.png` | Cabecera de login + favicon (sobre fondo claro) |
| `logo_horizontal.png` | Lockup horizontal (disponible para cabeceras anchas) |
| `logo_principal.png` | Versión principal (color, sobre claro) |
| `logo_negativo.png` | Versión negativa (sobre oscuro) |
| `logo_oscuro.png` | Versión oscura |
| `logo_contorno.png` | Solo contorno |

Favicon: `src/app/icon.png` (regenéralo o reemplázalo por la marca oficial).

Reglas: no deformar, rotar, recolorear fuera de paleta ni añadir sombras al logo.
Tamaño mínimo 40px y área de respeto ≈ ½ del diámetro del disco.
